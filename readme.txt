=====사용법=====
프로그램을 켜고 검색창에 검색하고 싶은 회사명을 입력한 후 밑에 해당 키워드를 가진 회사들이 뜨는데, 연봉 정보를 보고 싶은 회사를 리스트에서 클릭하면 됩니다.

=======설치 라이브러리 목록========
BeautifulSoup - html 데이터 받기 위해 사용
fuzzywuzzy - 회사 이름의 유사도를 따지기 위해 사용
python-Levenshtein - 수정하지 못한 오류에서 확인

==오류==
fuzzywuzzy 설치 및 사용 시 python-Levenshtein 패키지를 설치하라는 문구가 뜨는데 제 컴퓨터에서는 설치가 되지 않았습니다. 하지만 프로그램은 문제 없이 실행되었습니다. 대신 프로그램 실행 시 이 패키지가 없다는 Warning 문구가 떴는데, 프로그램 로직 상의 문제가 아니라고 생각되어 이 상태로 제출합니다.
